## ABOUT ME

Building projects, solving problems, and learning something new every day.

Greetings! I'm **Mansi Gole**, a Second-Year Electronics & Computer Science Engineering student from India with a strong interest in software development and modern web technologies.

My journey into tech began with curiosity about how applications work behind the scenes, which gradually led me toward Full-Stack Web Development. I enjoy transforming ideas into practical solutions and continuously improving my skills through hands-on projects and real-world problem solving.

Currently, I am focused on strengthening my development fundamentals while building projects that combine clean design, functionality, and meaningful impact.

---

## WHAT I DO

### Full-Stack Development
I build responsive web applications using React, JavaScript, Node.js, and Express.js. I enjoy working across both frontend and backend to create complete user experiences.

### Frontend Development
Creating intuitive and responsive interfaces is something I genuinely enjoy. I work with React, Tailwind CSS, Bootstrap, HTML, and CSS to build modern web applications.

### Project Building
I learn best by building. Whether it's a healthcare-focused platform like VitaLink, a Web3 wallet verification tool, or personal portfolio projects, I enjoy turning concepts into working products.

### Continuous Learning
Technology evolves rapidly, and I enjoy exploring new tools, frameworks, and development practices. Currently, I'm focused on improving my Full-Stack Development skills and strengthening my understanding of Data Structures & Algorithms.

### Problem Solving
I am particularly interested in building applications that solve real-world problems and create meaningful user experiences.
